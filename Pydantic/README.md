# Python Pydantic Projects

This repository contains beginner-to-intermediate projects demonstrating data validation and data modeling using **Pydantic**.

Pydantic is widely used in FastAPI, backend systems, APIs, data engineering, machine learning services, and production Python applications.

---

# Projects Included

## 1. User Model

Create and validate user information.

### Features

- Validate username
- Validate email
- Validate age
- Validate phone number
- Password length validation

Topics Covered

- BaseModel
- Field
- Email Validation
- Field Constraints
- Model Validation

---

## 2. Product Model

Validate product information before storing it.

### Features

- Product name validation
- Positive price validation
- Stock validation
- Category validation
- Discount calculation

Topics Covered

- BaseModel
- Field
- Numeric Constraints
- Computed Properties
- Validation Errors

---

## 3. Order Model

Simulate a simple e-commerce order.

### Features

- Customer validation
- Product validation
- Quantity validation
- Automatic total calculation
- Order summary

Topics Covered

- Nested Models
- BaseModel
- Field Validation
- Model Relationships
- Data Serialization
